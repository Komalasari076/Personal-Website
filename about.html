    • Hello, I am Siti Komalasari, a Fullstack Web Developer

    After deciding to take a gap year, I have used this time to study at a bootcamp and develop my skills in web development. Currently, I am working as a freelancer, handling various projects that allow me to apply and continuously sharpen my technical abilities.

    Selama di bootcamp, aku belajar banyak hal, mulai dari front-end development seperti HTML, CSS, dan JavaScript, hingga back-end development menggunakan Node.js dan Express. Aku juga mempelajari cara kerja database dengan MongoDB dan terbiasa bekerja dengan alat-alat seperti Git dan GitHub untuk version control.

    During the bootcamp, I learned a lot of things, from front-end development such as HTML, CSS, and JavaScript, to back-end development using Node.js and Express. I also studied how databases work with MongoDB and became familiar with tools like Git and GitHub for version control.

    Not only focused on coding, I also delved into UI/UX design to understand how to create user-friendly interfaces and provide the best experience for users. This combination of technical skills and design understanding has given me a strong foundation to contribute to various web development projects.
