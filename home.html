    What I Do as a Fullstack Web Developer

    Frontend Development

    • Build responsive user interfaces (UI) using HTML, CSS, and JavaScript.
    • Learn frontend frameworks like React to create dynamic and interactive UI components.
    • Design appealing and functional websites, ensuring they perform well across various devices.

    Backend Development

    • Learn the basics of backend development using Node.js and Express.
    • Manage simple databases using technologies like MongoDB or MySQL.
    • Create APIs that connect with the frontend to retrieve and send data.

    Experience with UI/UX Design

    • Learn the basics of UI/UX design, including creating wireframes and mockups.
    • Use design tools like Figma to build prototypes and design intuitive user experiences.
    • Conduct testing and iterate on designs based on feedback from users or mentors.

    Team Collaboration

    • Work collaboratively in small teams for projects, including task distribution in both frontend and backend development.
    • Communicate effectively with team members to successfully complete projects.
    • Use collaboration tools like Git and GitHub for version control and project management.

    Education and Training

    • Participate in a fullstack web development bootcamp to gain practical knowledge and skills in web development.
    • Work on real-world projects as part of the training, covering the entire development cycle from planning to implementation.
    • Receive guidance from mentors and engage in code reviews to improve code quality.
